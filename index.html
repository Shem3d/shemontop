Technique Scam via Apple et TikTok (By Shem)

Prérequis

Pour réaliser cette arnaque, il faut disposer de :

  • Un iPhone
  • Un compte TikTok avec au minimum 500 abonnés
  • Un compte PayPal
  • Une bonne excuse pour obtenir le numéro de téléphone de la victime

Obtenir le numéro de téléphone de la victime

Étape 1 : Rejoignez un serveur Discord fréquenté par des jeunes, par exemple « Blox Fruits FR » (lien : .gg/bloxfruitsfr).

Étape 2 : Trouvez une excuse pour demander le numéro de téléphone de votre cible.
Par exemple :
« Je bug sur Discord, viens on parle sur WhatsApp ».
Une fois que la victime accepte, vous obtenez son numéro de téléphone.

Utiliser le numéro pour le paiement via facture mobile

Étape 1 : Sur l’iPhone, allez dans Réglages > Paiement et livraisons > Mode de paiement

Étape 2 : Choisissez le mode de paiement Facture Mobile.

Étape 3 : Entrez le numéro de téléphone de la victime.
Un code de confirmation est alors envoyé à ce numéro.

Récupérer le code de confirmation

Deux méthodes pour obtenir ce code :

1ère méthode : Partage d’écran

Étape 1 : Demandez à la victime de faire un partage d’écran via TikTok, Snapchat ou Instagram, avec une excuse du type « Montre-moi tes pourtoi ».

Étape 2 : Dès que vous entrez le numéro pour le paiement via forfait mobile, la victime reçoit le code en notification, que vous pouvez voir grâce au partage d’écran.

2ème méthode : Demander directement le code

Étape 1 : Faites pression sur la victime en lui demandant rapidement le code, par exemple :

« Donne-moi le code vite, il reste 30 secondes, dépêche »

Transformer l’argent volé en argent réel

Étape 1 : Utilisez un compte TikTok avec au moins 500 abonnés pour pouvoir faire des lives.

Étape 2 : Payez des pièces TikTok avec le forfait mobile frauduleux sur un compte secondaire.

Étape 3 : Faites un live sur votre compte principal et envoyez-vous ces pièces depuis le compte secondaire.

Étape 4 : Une fois les pièces reçues, vous pouvez retirer l’argent via le solde TikTok vers PayPal ou un autre moyen.

(Avec le Forfait Mobile, vous pouvez aussi simplement payez sur des applications tel que Brawl Stars, Roblox, Snapchat, Discord sans forcément passer par TikTok sachant que la méthode avec TikTok est utile seulement pour convertir l’argent volé en argent sur PayPal.)
